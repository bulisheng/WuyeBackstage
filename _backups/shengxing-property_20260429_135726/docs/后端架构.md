# 物业小程序后台管理架构

## 1. 登录与权限

### 1.1 登录类型

- 超级管理员
- 管理员
- 客服
- 维修师傅
- 财务

### 1.2 权限说明

| 角色 | 权限范围 |
|---|---|
| 超级管理员 | 全部小区、全部功能、系统配置 |
| 管理员 | 指定小区的业务管理 |
| 客服 | 投诉建议、在线客服、业主咨询 |
| 维修师傅 | 报修工单、处理记录、维修进度 |
| 财务 | 缴费账单、催缴、缴费记录 |

## 4. 业主管理

### 4.1 业主列表

- 姓名
- 手机号
- 绑定房屋
- 所属小区
- 认证状态
- 注册时间
- 最近登录时间

### 4.2 业主认证

业主提交认证  
→ 后台审核  
→ 绑定手机号  
→ 绑定房屋  
→ 审核通过  
→ 开通业主服务权限

### 4.3 登录规则

- 只有业主提供手机号后才能登录
- 手机号需与业主档案/认证信息关联
- 未认证用户只能访问基础页面
- 已认证用户可访问缴费、报修、投票等业务

### 4.4 操作能力

- 查看业主详情
- 审核认证
- 绑定/解绑房屋
- 修改手机号
- 禁用/启用账号
- 查看服务记录

## 13. 通知配置 ⭐

### 13.1 通知配置入口

选择小区  
→ 机器人配置  
→ 场景配置  
→ 测试发送  
→ 启用通知

---

### 13.2 钉钉机器人配置

| 字段 | 说明 |
|---|---|
| 所属小区 | 当前配置生效的小区 |
| 机器人名称 | 如报修机器人、投诉机器人 |
| 是否启用 | 开启/关闭 |
| Webhook | 钉钉机器人地址 |
| Secret | 加签密钥 |
| 是否加签 | 是/否 |
| 默认@人员 | 手机号或钉钉用户 |
| 接收群 | 财务群/维修群/客服群 |
| 备注 | 配置说明 |

### 13.3 场景配置

| 场景 | 默认接收人 | 触发条件 |
|---|---|---|
| 报修通知 | 维修主管/维修群 | 用户提交报修 |
| 报修超时 | 工程主管 | 超过 SLA 未处理 |
| 投诉通知 | 客服/客服主管 | 用户提交投诉 |
| 投诉超时 | 物业主管 | 超时未回复 |
| 订单通知 | 商城运营 | 用户下单 |
| 转人工客服 | 客服人员 | 用户点击转人工 |
| 缴费逾期 | 财务人员 | 超过截止日期未缴费 |
| 物业服务 | 物业服务人员 | 用户提交服务申请 |
| 服务超时 | 物业主管 | 超时未接单 |
| 系统告警 | 技术/管理员 | 支付或接口异常 |

### 13.4 测试按钮

- 发送测试消息
- 测试 Webhook 是否可用
- 测试 Secret 签名是否正确
- 测试 @人员是否生效
- 记录测试日志
