# 使用者文档

这份文档给物业人员、运营人员和日常使用后台的人看。

## 你能用什么

### 微信小程序
- 登录和房屋绑定
- 看首页公告、待缴账单、报修状态
- 提交报修
- 提交投诉/表扬
- 看访客、装修、快递、代买菜等入口

### Web 管理台
- 公告管理
- 账单管理
- 报修管理
- 投诉/表扬/反馈管理
- 访客管理
- 装修管理
- 住户账号管理
- 房屋档案管理
- 物业人员管理
- 小区管理
- 快递和蔬菜商品/订单管理

### 多小区怎么用
- 每个小区都有自己的“项目名称”“默认负责人”和“功能开关”。
- 切换当前小区后，后台菜单和小程序入口会跟着变化。
- 小区里没开启的功能，前端不会显示入口。

## 怎么用 Web 管理台

1. 打开浏览器访问 Web 管理台。
2. 输入 API 地址和管理员密钥。
3. 登录后，从左侧菜单选择模块。
4. 在列表里搜索、筛选、排序。
5. 点某条记录查看详情、编辑或删除。
6. 支持批量操作的模块可以直接批量处理。

## 怎么设负责人和飞书通知人

- `小区管理` 里设每个小区自己的默认主管和主管列表。
- `物业人员管理` 里给人员绑定飞书成员。
- `投诉规则` 里设负责人和飞书通知人。
- 只有绑定了 `feishuUserId` 的人员，才有机会被真正 `@` 到。
- `当前启用功能` 可以直接看这个项目开了哪些模块。

## 怎么看投诉流程

1. 住户提交投诉。
2. 后端写入投诉队列。
3. 系统做 AI 分析。
4. 再推到飞书群。
5. 如果飞书成员已绑定，就会尽量做真正的 `@`。

## 常见注意事项

- `负责人` 只是规则负责人，不等于真正被通知的人。
- `飞书通知人` 才是飞书推送对象。
- 如果还没绑飞书标识，就只能发文字提醒。
- 多小区场景下，先切到正确的小区再配规则。
- 表扬和投诉已经统一进反馈管理，不要再去找单独的“表扬”老页面。
