# 社区物业小程序功能闭环

> 本文档描述核心业务从入口到处理、通知、归档的完整闭环。做新模块时先对照这里看是否能闭合。

> 当前进度：第一阶段已经完成；第二阶段 P1 正在推进中，缴费系统已开始落地，统一工单底座也已同步加入。

## 1. 首页闭环

进入首页  
→ 识别用户绑定小区  
→ 展示对应小区首页内容  
→ 点击高频入口  
→ 进入对应服务模块  
→ 完成服务/查看进度  
→ 回到首页形成复访

### 首页入口

- 生活缴费
- 报事报修
- 投诉建议
- 物业服务
- 盛兴严选
- 在线客服
- Banner 公告/活动
- TabBar：首页、生活、报修、我的

---

## 2. 业主认证闭环

用户进入小程序  
→ 提交业主认证信息  
→ 进入个人中心 绑定小区/楼栋/房屋  
→ 后台审核  
→ 审核通过  
→ 解锁缴费、报修、投票等业主服务  
→ 支持绑定多套房屋

---

## 3. 缴费闭环

后台生成账单  
→ 用户查看待缴账单  
→ 查看账单明细  
→ 微信支付  
→ 支付成功  
→ 更新账单状态  
→ 生成缴费记录  
→ 用户查看已缴账单

### 逾期分支

超过截止日期  
→ 标记 overdue  
→ 小程序提醒  
→ 短信催缴  
→ 钉钉提醒财务  
→ 电话催缴任务  
→ 记录 reminder_logs  
→ 用户完成支付

当前实现进度：
- 后台已经能维护账单、催缴和删除账单
- 住户侧已经能看到待缴账单并发起支付落库
- 支付记录表和催缴记录表已经开始使用

---

## 4. 报事报修闭环

用户提交报修  
→ 填写分类、描述、图片、预约时间  
→ 钉钉通知维修/客服  
→ 后台接单  
→ 分配维修人员  
→ 维修人员处理  
→ 更新处理记录时间线  
→ 状态变更通知用户  
→ 用户确认完成  
→ 用户评价  
→ 工单归档

### 异常分支

超时未处理  
→ 触发 SLA 提醒  
→ 钉钉通知负责人/主管  
→ 重新分配或升级处理

---

## 5. 投诉建议闭环

用户提交投诉/建议  
→ 可选择匿名  
→ 上传图片/说明  
→ 钉钉机器人 @客服  
→ 后台受理  
→ 分配负责人  
→ 负责人回复/处理  
→ 用户查看进度  
→ 问题解决  
→ 用户确认  
→ 用户评价  
→ 投诉归档

### 异常分支

超时未回复  
→ 自动提醒负责人  
→ 钉钉通知主管  
→ 升级处理

---

## 6. 物业服务闭环

用户选择服务  
→ 填写预约时间和服务内容  
→ 提交服务申请  
→ 钉钉通知物业服务人员  
→ 后台收到服务单  
→ 分配工作人员  
→ 工作人员执行  
→ 更新服务状态  
→ 用户确认完成  
→ 服务记录归档

### 服务类型

- 快递代寄
- 药品代取

### 通知节点

- 新服务申请 → 钉钉通知物业服务群
- 服务超时未接单 → 钉钉通知主管
- 服务完成 → 小程序通知用户

---

## 7. 商城闭环

用户浏览商品  
→ 查看商品详情  
→ 选择商品和数量  
→ 下单  
→ 微信支付  
→ 后台收到新订单  
→ 钉钉通知商城运营  
→ 备货/发货  
→ 用户确认收货  
→ 订单完成

### 可选分支

订单异常  
→ 后台处理  
→ 退款/售后  
→ 订单关闭

---

## 8. 在线客服闭环

用户提问  
→ FAQ 关键词匹配  
→ 命中则直接返回答案  
→ 未命中调用 AI  
→ 用户点击转人工  
→ 创建客服工单  
→ 钉钉通知 @客服  
→ 客服回复  
→ 用户确认解决  
→ 工单关闭  
→ 沉淀为 FAQ

---

## 9. 投票闭环

后台创建投票  
→ 设置单选/多选  
→ 设置一人一票或一户一票  
→ 指定参与范围  
→ 用户查看投票  
→ 系统校验投票资格  
→ 用户提交投票  
→ 后台统计结果  
→ 公示结果  
→ 投票归档

---

## 10. 社区活动闭环

后台发布活动  
→ 设置报名时间和名额限制  
→ 用户报名  
→ 报名成功提醒  
→ 生成报名记录/签到二维码  
→ 活动现场扫码签到  
→ 后台核销  
→ 活动结束  
→ 统计报名和签到数据  
→ 活动归档

---

## 11. 我的闭环

用户进入我的  
→ 查看业主认证状态  
→ 管理我的房屋  
→ 查看我的报修  
→ 查看我的订单  
→ 查看我的缴费  
→ 查看投诉/客服/活动记录  
→ 形成个人服务中心闭环

---

## 12. 全局通知闭环

业务动作触发  
→ 判断通知类型  
→ 小程序消息/短信/钉钉/电话任务  
→ 发送通知  
→ 记录通知日志  
→ 失败重试  
→ 后台可追踪

---

## 13. 总体业务闭环

业主认证  
→ 使用服务  
→ 支付/报修/投诉/购物/活动/投票  
→ 后台处理  
→ 状态通知  
→ 用户确认  
→ 评价反馈  
→ 数据沉淀  
→ 运营优化
