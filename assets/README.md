# 物业管理系统资源文件说明

## 需要添加的TabBar图标

请在 `assets` 目录下添加以下图片文件：

### TabBar图标（建议尺寸：81x81px，格式：png）

| 文件名 | 说明 |
|--------|------|
| `tab-home.png` | 首页-未选中 |
| `tab-home-active.png` | 首页-选中 |
| `tab-fee.png` | 缴费-未选中 |
| `tab-fee-active.png` | 缴费-选中 |
| `tab-repair.png` | 报修-未选中 |
| `tab-repair-active.png` | 报修-选中 |
| `tab-my.png` | 我的-未选中 |
| `tab-my-active.png` | 我的-选中 |
| `default-avatar.png` | 默认头像 |

## 其他资源
- 可以使用 iconfont 或自行设计图标
- 图标颜色建议与主题色 #1890ff 保持一致
